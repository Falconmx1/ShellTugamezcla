# ShellTugamezcla
ShellTuga is a powerful and intuitive graphical front-end for The Sleuth Kit (TSK), inspired by Autopsy but focused on simplicity and speed. Designed for digital forensics analysts, ShellTuga replaces complex command-line inputs with an organized GUI featuring a turtle-powered forensic browser. Supports Windows (NTFS, FAT) and UNIX (EXT2/3/4, FFS)
